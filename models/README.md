# Local Models

Put externally hosted Gemma `.litertlm` files under `litert/` when needed for
local runs.

Expected LiteRT paths:

```text
litert/gemma-4-E2B-it-web.litertlm
litert/gemma-4-E2B-it.litertlm
```

Actual model binaries, partial downloads, and split model chunks are ignored by
Git and must not be committed.
